# ✍️ MgBlog

A modern, clean, and responsive **personal blog website** built with **Python (Flask)**, **SQLite**, and **Tailwind CSS**.

![MgBlog Preview](preview.png)

## ✨ Features

- Beautiful homepage with latest posts
- Full blog post reading page
- Admin panel to create, edit, and delete posts
- Category system (Tech, Programming, Lifestyle, etc.)
- Responsive design (works perfectly on mobile)
- Dark/Light mode support
- Clean and modern UI/UX design
- Search functionality (coming soon)

## 🛠️ Tech Stack

- **Backend**: Python + Flask
- **Database**: SQLite
- **Frontend**: HTML + Tailwind CSS + JavaScript
- **Templating**: Jinja2
